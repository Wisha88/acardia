商户进件 特约商户入网（产品介绍） 服务商拓展特约商户入驻拉卡拉接口
特约商户入网（API接口）
基础支付 被扫支付（API接口） 商户通过调用拉卡拉提供的被扫支付api接口，在支付场景中调起拉卡拉支付模块完成收款
公众号支付（API接口） 商户通过在自己的公众号中调用拉卡拉主扫支付接口完成线上交易收款流程
小程序支付（API接口） 商户通过在自己的小程序中调用拉卡拉主扫支付接口完成线上交易收款流程
收款码支付（API接口） 商户通通过调用拉卡拉收款码接口，即可生成付款吗，用户通过扫描付款吗即可完成收款
数币支付（API接口） 拉卡拉所有交易场景接口均支持数币支付场景
B2B B2B转账支付 转账支付是机构/商户向拉卡拉提交申请动态账号预下单，拉卡拉根据动态账号生成规则生成账号并反馈给机构/商户，机构/商户向用户展示动态账户信息，用户获取账户信息后通过银行APP完成转账交易。
B2B货到付款 货到付款是快递公司把商品送到您手上以后，您检查过商品没任何问题后，通过购买平台确认收到商品。购买平台将收款订单推送至快递员携带收款工具，通过刷卡或扫码等支付方式完成付款的流程。
B2B订单收银台 订单收银台是机构/商户向拉卡拉提交申请订单交易下单，拉卡拉根据订单信息生成订单收银台URL反馈给机构/商户，机构/商户向用户展示收银台地址或动态二维码，用户打开订单收银台根据选择微信、支付宝、云闪付、转账、POS、数币支付。
新零售 新零售项目方案 为大B商户、机构、ISV服务商提供新零售场景下的多种收付款服务
资金应用 分账（产品介绍） 服务商分账，主要用于服务商帮助特约商户完成订单收单成功后的资金分配。
分账（API接口-开通）
分账（API接口-交易）
资金结算 资金结算 服务商调用拉卡拉结算接口，拉卡拉服务商完成后续对商户结算划款流程
经营能力 商户交易对账 拉卡拉可为服务商或商户提供定制化的对账单，满足商户对账需求
